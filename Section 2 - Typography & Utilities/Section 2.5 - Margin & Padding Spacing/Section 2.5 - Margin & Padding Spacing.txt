Section 2.5 Margin & Padding Spacing:

--------------------------------------
Margin & Padding:
--------------------------------------
- Bootstrap have margin and padding classes to add spacing to to your html elements. This is great as you do not need to use markup to add spacing for example <br> etc. Also you do not need to go into your custom CSS to add custom margin and paddings.

- We can add margin/padding top or bottom, between a range of 0-5, using the following classes:
   Margin bottom     Margin top     Padding bottom     Padding top
   mb-0              mt-0           pb-0               pt-0
   mb-1              mt-1           pb-1               pt-1
   mb-2              mt-2           pb-2               pt-2
   mb-3              mt-3           pb-3               pt-3
   mb-4              mt-4           pb-4               pt-4
   mb-5              mt-5           pb-5               pt-5

- Margin is the border outside the element container (represented in green) whereas the padding is the border inside the element container (represented in yellow). We used bg-success class to colour the background green and bg-warning to colour the background yellow to visually represent the html element's container so that we can visually see the affects of the margin and padding spacing classes within bootstrap.

- We can add margin/padding left or right, between a range of 0-5, using the following classes:
   Margin left       Margin right   Padding left       Padding right
   ml-0              mr-0           pl-0               pr-0
   ml-1              mr-1           pl-1               pr-1
   ml-2              mr-2           pl-2               pr-2
   ml-3              mr-3           pl-3               pr-3
   ml-4              mr-4           pl-4               pr-4
   ml-5              mr-5           pl-5               pr-5

- We can add margin/padding on both the top and bottom, between a range of 0-5, using the following classes:
   Margin top/bottom       Margin top/bottom
   my-0                    py-0
   my-1                    py-1
   my-2                    py-2
   my-3                    py-3
   my-4                    py-4
   my-5                    py-5

- We can add margin/padding on both the left and right, between a range of 0-5, using the following classes:
   Margin left/Right       Margin right
   mx-0                    px-0
   mx-1                    px-1
   mx-2                    px-2
   mx-3                    px-3
   mx-4                    px-4
   mx-5                    px-5

- We can add margin/padding on all sides, between a range of 0-5, using the following classes:
   Margin all sides        Margin all sides
   m-0                     p-0
   m-1                     p-1
   m-2                     p-2
   m-3                     p-3
   m-4                     p-4
   m-5                     p-5

- We can add horizontal centering of a html element (same as margin-left = auto and margin-right = auto) by using the following class:
   mx-auto


Please refer to the 2_5_spacing.html markup within the bootstrap_sandbox folder for examples of the above. To view what these would look like open the html file in your browser.
